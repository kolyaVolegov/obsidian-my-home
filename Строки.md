2. –ü–æ–¥—Ä–æ–±–Ω–æ –æ–ø–∏—à–∏—Ç–µ –º–µ—Ö–∞–Ω–∏–∑–º String Pool –∏ –ø—Ä–æ—Ü–µ—Å—Å –∏–Ω—Ç–µ—Ä–Ω–∏—Ä–æ–≤–∞–Ω–∏—è —Å—Ç—Ä–æ–∫ –≤ Java. –û–±—ä—è—Å–Ω–∏—Ç–µ, –∫–∞–∫ –∏–∑–º–µ–Ω–µ–Ω–∏–µ —Ä–∞—Å–ø–æ–ª–æ–∂–µ–Ω–∏—è String Pool –≤ JVM (–¥–æ –∏ –ø–æ—Å–ª–µ Java 7) –ø–æ–≤–ª–∏—è–ª–æ –Ω–∞ —É–ø—Ä–∞–≤–ª–µ–Ω–∏–µ –ø–∞–º—è—Ç—å—é –∏ –≤–æ–∑–º–æ–∂–Ω—ã–µ –æ—à–∏–±–∫–∏ OutOfMemoryError.
3. –°—Ä–∞–≤–Ω–∏—Ç–µ –∏ –ø—Ä–æ—Ç–∏–≤–æ–ø–æ—Å—Ç–∞–≤—å—Ç–µ –º–µ—Ç–æ–¥—ã equals(), equalsIgnoreCase(), compareTo(), compareToIgnoreCase() –∏ –æ–ø–µ—Ä–∞—Ç–æ—Ä == –ø—Ä–∏–º–µ–Ω–∏—Ç–µ–ª—å–Ω–æ –∫ —Å—Ä–∞–≤–Ω–µ–Ω–∏—é —Å—Ç—Ä–æ–∫ –≤ Java. –û–±—ä—è—Å–Ω–∏—Ç–µ, –∫–æ–≥–¥–∞ –∏ –ø–æ—á–µ–º—É —Å–ª–µ–¥—É–µ—Ç –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –∫–∞–∂–¥—ã–π –∏–∑ –Ω–∏—Ö, –∏ –∫–∞–∫–∏–µ –ø–æ–¥–≤–æ–¥–Ω—ã–µ –∫–∞–º–Ω–∏ —Å—É—â–µ—Å—Ç–≤—É—é—Ç –ø—Ä–∏ –Ω–µ–ø—Ä–∞–≤–∏–ª—å–Ω–æ–º –≤—ã–±–æ—Ä–µ.
4. –û–±—ä—è—Å–Ω–∏—Ç–µ –∫–æ–Ω—Ü–µ–ø—Ü–∏—é —Ä–µ–≥—É–ª—è—Ä–Ω—ã—Ö –≤—ã—Ä–∞–∂–µ–Ω–∏–π –≤ Java, –ø–æ–¥—Ä–æ–±–Ω–æ –æ–ø–∏—Å—ã–≤–∞—è —Ä–æ–ª—å –∫–ª–∞—Å—Å–æ–≤ Pattern –∏ Matcher. –ü—Ä–æ–¥–µ–º–æ–Ω—Å—Ç—Ä–∏—Ä—É–π—Ç–µ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ —Ä–∞–∑–ª–∏—á–Ω—ã—Ö —Ç–∏–ø–æ–≤ –∫–≤–∞–Ω—Ç–∏—Ñ–∏–∫–∞—Ç–æ—Ä–æ–≤ (–∂–∞–¥–Ω—ã—Ö, —Å–≤–µ—Ä—Ö–∂–∞–¥–Ω—ã—Ö, –ª–µ–Ω–∏–≤—ã—Ö) –∏ –ø—Ä–æ–∏–ª–ª—é—Å—Ç—Ä–∏—Ä—É–π—Ç–µ –∏—Ö –ø–æ–≤–µ–¥–µ–Ω–∏–µ –Ω–∞ –∫–æ–Ω–∫—Ä–µ—Ç–Ω—ã—Ö –ø—Ä–∏–º–µ—Ä–∞—Ö.
5. –†–∞—Å—Å–º–æ—Ç—Ä–∏—Ç–µ, –∫–∞–∫ Java (–Ω–∞—á–∏–Ω–∞—è —Å –≤–µ—Ä—Å–∏–∏ 9) –æ–ø—Ç–∏–º–∏–∑–∏—Ä—É–µ—Ç —Ö—Ä–∞–Ω–µ–Ω–∏–µ —Å—Ç—Ä–æ–∫ —Å –ø–æ–º–æ—â—å—é "–∫–æ–º–ø–∞–∫—Ç–Ω—ã—Ö —Å—Ç—Ä–æ–∫" –∏ –∫–∞–∫ String Deduplication –≤–ª–∏—è–µ—Ç –Ω–∞ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ –ø–∞–º—è—Ç–∏. –û–±—ä—è—Å–Ω–∏—Ç–µ, –∫–∞–∫ —ç—Ç–∏ –¥–≤–µ —Ñ—É–Ω–∫—Ü–∏–∏ —Ä–∞–±–æ—Ç–∞—é—Ç –≤–º–µ—Å—Ç–µ –¥–ª—è –ø–æ–≤—ã—à–µ–Ω–∏—è —ç—Ñ—Ñ–µ–∫—Ç–∏–≤–Ω–æ—Å—Ç–∏ —É–ø—Ä–∞–≤–ª–µ–Ω–∏—è –ø–∞–º—è—Ç—å—é –≤ JVM.

–ì–ª–æ—Å—Å–∞—Ä–∏–π –∫–ª—é—á–µ–≤—ã—Ö —Ç–µ—Ä–º–∏–Ω–æ–≤

- String: –ù–µ–∏–∑–º–µ–Ω—è–µ–º—ã–π (immutable) –∫–ª–∞—Å—Å –≤ Java, –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è—é—â–∏–π –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç—å —Å–∏–º–≤–æ–ª–æ–≤. –õ—é–±–∞—è –æ–ø–µ—Ä–∞—Ü–∏—è, –∏–∑–º–µ–Ω—è—é—â–∞—è —Å—Ç—Ä–æ–∫—É, —Ñ–∞–∫—Ç–∏—á–µ—Å–∫–∏ —Å–æ–∑–¥–∞–µ—Ç –Ω–æ–≤—É—é —Å—Ç—Ä–æ–∫—É.
- StringBuffer: –ò–∑–º–µ–Ω—è–µ–º—ã–π (mutable) –∫–ª–∞—Å—Å –≤ Java, –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è—é—â–∏–π –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç—å —Å–∏–º–≤–æ–ª–æ–≤. –ü–æ—Ç–æ–∫–æ–±–µ–∑–æ–ø–∞—Å–µ–Ω, —Ç–∞–∫ –∫–∞–∫ –≤—Å–µ –µ–≥–æ –º–µ—Ç–æ–¥—ã —Å–∏–Ω—Ö—Ä–æ–Ω–∏–∑–∏—Ä–æ–≤–∞–Ω—ã. –ò—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –¥–ª—è –º–∞–Ω–∏–ø—É–ª—è—Ü–∏–π —Å–æ —Å—Ç—Ä–æ–∫–∞–º–∏ –≤ –º–Ω–æ–≥–æ–ø–æ—Ç–æ—á–Ω—ã—Ö —Å—Ä–µ–¥–∞—Ö.
- StringBuilder: –ò–∑–º–µ–Ω—è–µ–º—ã–π (mutable) –∫–ª–∞—Å—Å –≤ Java, –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è—é—â–∏–π –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç—å —Å–∏–º–≤–æ–ª–æ–≤. –ù–µ —Å–∏–Ω—Ö—Ä–æ–Ω–∏–∑–∏—Ä–æ–≤–∞–Ω, –ø–æ—ç—Ç–æ–º—É –Ω–µ –ø–æ—Ç–æ–∫–æ–±–µ–∑–æ–ø–∞—Å–µ–Ω, –Ω–æ —Ä–∞–±–æ—Ç–∞–µ—Ç –±—ã—Å—Ç—Ä–µ–µ, —á–µ–º StringBuffer –≤ –æ–¥–Ω–æ–ø–æ—Ç–æ—á–Ω—ã—Ö —Å—Ä–µ–¥–∞—Ö.
- –ò–º–º—É—Ç–∞–±–µ–ª—å–Ω–æ—Å—Ç—å (Immutable): –°–≤–æ–π—Å—Ç–≤–æ –æ–±—ä–µ–∫—Ç–∞, –æ–∑–Ω–∞—á–∞—é—â–µ–µ, —á—Ç–æ –µ–≥–æ —Å–æ—Å—Ç–æ—è–Ω–∏–µ –Ω–µ –º–æ–∂–µ—Ç –±—ã—Ç—å –∏–∑–º–µ–Ω–µ–Ω–æ –ø–æ—Å–ª–µ —Å–æ–∑–¥–∞–Ω–∏—è. –û–±—ä–µ–∫—Ç—ã –∫–ª–∞—Å—Å–∞ String –≤ Java —è–≤–ª—è—é—Ç—Å—è –Ω–µ–∏–∑–º–µ–Ω—è–µ–º—ã–º–∏.
- –ú—É—Ç–∞–±–µ–ª—å–Ω–æ—Å—Ç—å (Mutable): –°–≤–æ–π—Å—Ç–≤–æ –æ–±—ä–µ–∫—Ç–∞, –æ–∑–Ω–∞—á–∞—é—â–µ–µ, —á—Ç–æ –µ–≥–æ —Å–æ—Å—Ç–æ—è–Ω–∏–µ –º–æ–∂–µ—Ç –±—ã—Ç—å –∏–∑–º–µ–Ω–µ–Ω–æ –ø–æ—Å–ª–µ —Å–æ–∑–¥–∞–Ω–∏—è. –û–±—ä–µ–∫—Ç—ã –∫–ª–∞—Å—Å–æ–≤ StringBuffer –∏ StringBuilder —è–≤–ª—è—é—Ç—Å—è –∏–∑–º–µ–Ω—è–µ–º—ã–º–∏.
- –ö–æ–Ω–∫–∞—Ç–µ–Ω–∞—Ü–∏—è (Concatenation): –û–ø–µ—Ä–∞—Ü–∏—è –æ–±—ä–µ–¥–∏–Ω–µ–Ω–∏—è –¥–≤—É—Ö –∏–ª–∏ –±–æ–ª–µ–µ —Å—Ç—Ä–æ–∫ –≤ –æ–¥–Ω—É. –í Java –º–æ–∂–µ—Ç –±—ã—Ç—å –≤—ã–ø–æ–ª–Ω–µ–Ω–∞ —Å –ø–æ–º–æ—â—å—é –æ–ø–µ—Ä–∞—Ç–æ—Ä–∞ + –∏–ª–∏ –º–µ—Ç–æ–¥–æ–≤ concat(), join().
- String Pool (–ü—É–ª —Å—Ç—Ä–æ–∫): –°–ø–µ—Ü–∏–∞–ª—å–Ω–∞—è –æ–±–ª–∞—Å—Ç—å –ø–∞–º—è—Ç–∏ –≤ –∫—É—á–µ JVM, –≥–¥–µ —Ö—Ä–∞–Ω—è—Ç—Å—è —É–Ω–∏–∫–∞–ª—å–Ω—ã–µ —Å—Ç—Ä–æ–∫–æ–≤—ã–µ –ª–∏—Ç–µ—Ä–∞–ª—ã –¥–ª—è –æ–ø—Ç–∏–º–∏–∑–∞—Ü–∏–∏ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏—è –ø–∞–º—è—Ç–∏.
- –ò–Ω—Ç–µ—Ä–Ω–∏—Ä–æ–≤–∞–Ω–∏–µ (Interning): –ü—Ä–æ—Ü–µ—Å—Å –ø–æ–º–µ—â–µ–Ω–∏—è —Å—Ç—Ä–æ–∫–æ–≤–æ–≥–æ –ª–∏—Ç–µ—Ä–∞–ª–∞ –≤ String Pool. –ï—Å–ª–∏ —Å—Ç—Ä–æ–∫–∞ —Å —Ç–∞–∫–∏–º –∑–Ω–∞—á–µ–Ω–∏–µ–º —É–∂–µ —Å—É—â–µ—Å—Ç–≤—É–µ—Ç –≤ –ø—É–ª–µ, –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç—Å—è —Å—Å—ã–ª–∫–∞ –Ω–∞ —Å—É—â–µ—Å—Ç–≤—É—é—â–∏–π –æ–±—ä–µ–∫—Ç.
- intern(): –ú–µ—Ç–æ–¥ –∫–ª–∞—Å—Å–∞ String, –ø–æ–∑–≤–æ–ª—è—é—â–∏–π –≤—Ä—É—á–Ω—É—é –¥–æ–±–∞–≤–∏—Ç—å —Å—Ç—Ä–æ–∫—É –≤ String Pool –∏–ª–∏ –ø–æ–ª—É—á–∏—Ç—å —Å—Å—ã–ª–∫—É –Ω–∞ —É–∂–µ —Å—É—â–µ—Å—Ç–≤—É—é—â—É—é –≤ –ø—É–ª–µ —Å—Ç—Ä–æ–∫—É —Å —Ç–∞–∫–∏–º –∂–µ –∑–Ω–∞—á–µ–Ω–∏–µ–º.
- printf —Ñ–æ—Ä–º–∞—Ç–∏—Ä–æ–≤–∞–Ω–∏–µ: –°–∏–Ω—Ç–∞–∫—Å–∏—Å, –∏—Å–ø–æ–ª—å–∑—É–µ–º—ã–π –¥–ª—è —Ñ–æ—Ä–º–∞—Ç–∏—Ä–æ–≤–∞–Ω–Ω–æ–≥–æ –≤—ã–≤–æ–¥–∞ —Ç–µ–∫—Å—Ç–∞, –ø—Ä–∏–º–µ–Ω—è–µ–º—ã–π –≤ —Ä–∞–∑–ª–∏—á–Ω—ã—Ö —è–∑—ã–∫–∞—Ö –ø—Ä–æ–≥—Ä–∞–º–º–∏—Ä–æ–≤–∞–Ω–∏—è, –≤–∫–ª—é—á–∞—è Java (—á–µ—Ä–µ–∑ String.format() –∏ System.out.printf()).
- –°–ø–µ—Ü–∏—Ñ–∏–∫–∞—Ç–æ—Ä —Ñ–æ—Ä–º–∞—Ç–∞: –°–∏–º–≤–æ–ª –∏–ª–∏ –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç—å —Å–∏–º–≤–æ–ª–æ–≤, –∏—Å–ø–æ–ª—å–∑—É–µ–º—ã–µ –≤ —Ñ–æ—Ä–º–∞—Ç–∏—Ä–æ–≤–∞–Ω–Ω—ã—Ö —Å—Ç—Ä–æ–∫–∞—Ö (–Ω–∞–ø—Ä–∏–º–µ—Ä, %s, %d, %f) –¥–ª—è —É–∫–∞–∑–∞–Ω–∏—è —Ç–∏–ø–∞ –∏ —Ñ–æ—Ä–º–∞—Ç–∞ –≤—ã–≤–æ–¥–∏–º—ã—Ö –¥–∞–Ω–Ω—ã—Ö.
- –†–µ–≥—É–ª—è—Ä–Ω–æ–µ –≤—ã—Ä–∞–∂–µ–Ω–∏–µ (Regex/RegExp/RE): –ü–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç—å —Å–∏–º–≤–æ–ª–æ–≤, –∫–æ—Ç–æ—Ä–∞—è –æ–ø—Ä–µ–¥–µ–ª—è–µ—Ç —à–∞–±–ª–æ–Ω –ø–æ–∏—Å–∫–∞ –¥–ª—è —Å–æ–ø–æ—Å—Ç–∞–≤–ª–µ–Ω–∏—è —Å –¥—Ä—É–≥–∏–º–∏ —Å—Ç—Ä–æ–∫–∞–º–∏.
- Pattern (–∫–ª–∞—Å—Å): –ö–ª–∞—Å—Å –≤ –ø–∞–∫–µ—Ç–µ java.util.regex, –∫–æ—Ç–æ—Ä—ã–π –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è–µ—Ç —Å–æ–±–æ–π —Å–∫–æ–º–ø–∏–ª–∏—Ä–æ–≤–∞–Ω–Ω–æ–µ —Ä–µ–≥—É–ª—è—Ä–Ω–æ–µ –≤—ã—Ä–∞–∂–µ–Ω–∏–µ. –ò—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –¥–ª—è —Å–æ–∑–¥–∞–Ω–∏—è –æ–±—ä–µ–∫—Ç–æ–≤ Matcher.
- Matcher (–∫–ª–∞—Å—Å): –ö–ª–∞—Å—Å –≤ –ø–∞–∫–µ—Ç–µ java.util.regex, –∫–æ—Ç–æ—Ä—ã–π —è–≤–ª—è–µ—Ç—Å—è –¥–≤–∏–∂–∫–æ–º –¥–ª—è –≤—ã–ø–æ–ª–Ω–µ–Ω–∏—è –æ–ø–µ—Ä–∞—Ü–∏–π —Å–æ–ø–æ—Å—Ç–∞–≤–ª–µ–Ω–∏—è —Å –≤—Ö–æ–¥–Ω–æ–π —Å—Ç—Ä–æ–∫–æ–π –Ω–∞ –æ—Å–Ω–æ–≤–µ —Å–∫–æ–º–ø–∏–ª–∏—Ä–æ–≤–∞–Ω–Ω–æ–≥–æ —à–∞–±–ª–æ–Ω–∞ (Pattern).
- –ú–µ—Ç–∞—Å–∏–º–≤–æ–ª: –°–ø–µ—Ü–∏–∞–ª—å–Ω—ã–π —Å–∏–º–≤–æ–ª –≤ —Ä–µ–≥—É–ª—è—Ä–Ω—ã—Ö –≤—ã—Ä–∞–∂–µ–Ω–∏—è—Ö, –∫–æ—Ç–æ—Ä—ã–π –∏–º–µ–µ—Ç –æ—Å–æ–±–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ (–Ω–∞–ø—Ä–∏–º–µ—Ä, ., ^, $, *, +, ?).
- –ö–≤–∞–Ω—Ç–∏—Ñ–∏–∫–∞—Ç–æ—Ä: –°–∏–º–≤–æ–ª –∏–ª–∏ –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç—å —Å–∏–º–≤–æ–ª–æ–≤ –≤ —Ä–µ–≥—É–ª—è—Ä–Ω—ã—Ö –≤—ã—Ä–∞–∂–µ–Ω–∏—è—Ö, –∫–æ—Ç–æ—Ä—ã–µ —É–∫–∞–∑—ã–≤–∞—é—Ç, —Å–∫–æ–ª—å–∫–æ —Ä–∞–∑ –¥–æ–ª–∂–µ–Ω –ø–æ–≤—Ç–æ—Ä—è—Ç—å—Å—è –ø—Ä–µ–¥—à–µ—Å—Ç–≤—É—é—â–∏–π —ç–ª–µ–º–µ–Ω—Ç (–Ω–∞–ø—Ä–∏–º–µ—Ä, *, +, ?, {n}, {n,m}).
- –ñ–∞–¥–Ω—ã–π –∫–≤–∞–Ω—Ç–∏—Ñ–∏–∫–∞—Ç–æ—Ä (Greedy Quantifier): –ö–≤–∞–Ω—Ç–∏—Ñ–∏–∫–∞—Ç–æ—Ä, –∫–æ—Ç–æ—Ä—ã–π –ø–æ —É–º–æ–ª—á–∞–Ω–∏—é –ø—ã—Ç–∞–µ—Ç—Å—è –Ω–∞–π—Ç–∏ –º–∞–∫—Å–∏–º–∞–ª—å–Ω–æ –¥–ª–∏–Ω–Ω–æ–µ —Å–æ–≤–ø–∞–¥–µ–Ω–∏–µ.

- –°–≤–µ—Ä—Ö–∂–∞–¥–Ω—ã–π –∫–≤–∞–Ω—Ç–∏—Ñ–∏–∫–∞—Ç–æ—Ä (Possessive Quantifier): –ö–≤–∞–Ω—Ç–∏—Ñ–∏–∫–∞—Ç–æ—Ä, –∑–∞ –∫–æ—Ç–æ—Ä—ã–º —Å–ª–µ–¥—É–µ—Ç +, –∫–æ—Ç–æ—Ä—ã–π –≤–µ–¥–µ—Ç —Å–µ–±—è –∫–∞–∫ –∂–∞–¥–Ω—ã–π, –Ω–æ –Ω–µ –ø—Ä–æ–∏–∑–≤–æ–¥–∏—Ç —Ä–µ–≤–µ—Ä—Å–∏–≤–Ω—ã–π –ø–æ–∏—Å–∫ (backtracking).
